Reports

Once a file has been checked, selecting it in the file list will display its reports. Each file has a Validation report and an Information report. The Validation report may list the following kinds of problems:
Audio Format
Problems with the technical specifications of the MP3 file, such as bit rate or sampling frequency.
File Name
Problems with the file name not matching the LibriVox naming conventions.
Track Length
The duration is unusually short or long.
Volume Problems
The recording is too loud or quiet.
Background Noise
The recording has too much background noise.
Clipping Problems
The recording has an excessive amount of clipping.
DC Bias
The waveform is off-center.
ID3 Metadata
The ID3 tag is missing, the wrong version, or incomplete.

The Information report provides general information about the file, including the technical specs, the track length, volume, clipping, and DC bias measurements, and the file's basic ID3 metadata.
