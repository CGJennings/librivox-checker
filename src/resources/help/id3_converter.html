ID3 Tag Converter

You can upgrade the ID3 tags in your files automatically. Select the files you want to modify in the file table and choose the Tools|Convert ID3 Tags menu item.

In the dialog that appears, choose the tag version you want to switch the files to: either ID3 v2.3 or ID3 v2.4. Choosing Convert will begin the conversion. A dialog will appear that displays the result of the conversion attempt for each file. Checker will attempt to convert all of the fields in the original tag, even fields that it does not display in the Information report. If a file's tag already matches the target version, the tag will not be changed. After the conversion finishes, the selected files will be re-checked to update their reports with their new metadata information.
